mongoose-troop  ℵ

mongoose-troop is a utility package for Mongoose, composed of useful plugins and simple helpers.

The complete annotated source code is also available.

Installation

The project can be installed via NPM, or by cloning this repo into your project. 

    
  npm install mongoose-troop

or 

    
  git clone git://github.com/tblobaum/mongoose-troop.git
  cd mongoose-troop
  npm link
